# Privacy Policy - AntiBet Alerta

This repository contains the Privacy Policy page for the AntiBet Alerta application.

## 🌐 View Online

Access the page at: `https://brendaalcantara-liv.github.io/privacy-policy`

## 📋 How to Use

1. Clone this repository
2. Customize the content in `index.html` with your application information
3. Adjust colors and styles in `styles.css` if desired
4. Commit and push to GitHub
5. Enable GitHub Pages in repository settings

## 🚀 How to Host on GitHub Pages

1. Push this repository to GitHub
2. Go to repository Settings
3. Scroll to the "Pages" section
4. Under "Source", select the `main` branch (or `master`)
5. Select the `/ (root)` folder
6. Click "Save"
7. Wait a few minutes and your page will be available!

## ✏️ Customization

Edit the following items in the `index.html` file:

- Application name
- Last updated date
- Contact information (email and address)
- Specific sections according to your app needs

## 📱 Responsive

The page is fully responsive and works perfectly on mobile and desktop devices.
